Usage guidelines:<br>
compile with `make all` inside course_project directory <br>
nameserver: `./bin/name_server` 
<br>
storageserver: `./bin/storage_server <nameserver ip> <port to assign to storageserver> <storage server id>`
<br>
client: `./bin/client <nameserver ip>`

Use HELP command in client to view all commands available and their usage
